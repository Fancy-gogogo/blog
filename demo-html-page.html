这是HTML页面的标题

这段内容来自一个 .html 文件，而不是 .md 文件。它被 Hugo 直接渲染为博客文章，复用了博客的主题样式——导航栏、侧边栏、页脚都在。

你可以做什么

  • 使用任意 HTML 标签和结构
  • 嵌入 <style> 块自定义样式
  • 嵌入 <script> 添加交互
  • 使用 CSS 动画、Canvas、SVG 等

注意事项

  1. 只放 body 内的内容，不要粘贴完整的 HTML 文档
  2. 自定义样式用唯一类名包裹（如本页的 .demo-html-page），避免与主题冲突
  3. front matter 中的 description 字段要填，用于搜索和卡片预览
